## 使用小贴士
- 查看当前分支：`git branch --show-current`
- 查看简短提交历史：`git log --oneline -n 5`
- 返回主分支：`git checkout main`
- 同步远程 main 最新更改：`git pull`
- 推送当前分支并建立跟踪：`git push -u origin $(git branch --show-current)`# Hello Git
